Tranquility Now Code Demo

Built with: HTML, CSS, jQuery, D3.js, npm, Express.js, MongoDB

Manage your stress by recording the activities that calm you down. Some activities will relax you better than others. As individuals, we react differently to stimuli that can increase or decrease stress. Each user keeps a record of activites they believe decrease their stress. A lowered heart rate is an indication of a stimulus that calms the user.

Tasks and Treasures Code Demo

Built with: React, Redux, React Joyride, CSS, npm, Express.js, MongoDB, Passport and JWT

Parents create tasks for their kids to complete. Once the child finishes their tasks they get a prize. My brother wanted an app to track the chores he gave his girls, and to reward them when they finished their work. The app uses long polling for events that update the state - updates than can affect both parent and child users. Sometimes the updates are not automatic and each user needs to refresh the page.
